# ERP Lite SAP S/4HANA Simulation

## Overview

This project simulates a real-world SAP ERP implementation for a distribution company.

The objective is to design an ERP solution following SAP Activate methodology, from Discovery Workshop to Go-Live.

---

## Business Scenario

Company: Minh Phat Distribution

Industry: Consumer Goods Distribution

Business Size:

- 50 Employees
- 2 Branches
- 2 Warehouses
- 2,000+ SKUs

Current Systems:
