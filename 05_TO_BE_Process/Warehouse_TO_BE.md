# TO-BE Process Design

## Module

Warehouse Management

---

# Business Objective

Provide real-time inventory visibility and standardized warehouse operations through integrated inventory management.

---

# Future Process Flow

Sales Order Approved

↓

ERP Creates Delivery Request

↓

Warehouse Receives Pick List

↓

Warehouse Picks Products

↓

ERP Validates Batch

↓

ERP Validates Expiry Date

↓

Warehouse Confirms Goods Issue

↓

Inventory Updates Automatically

↓

Dashboard Updates in Real Time

# Process Improvements

| AS-IS                  | TO-BE                |
| ---------------------- | -------------------- |
| Excel Inventory        | Real-Time Inventory  |
| Phone Inquiry          | ERP Inventory Lookup |
| Manual Batch Tracking  | Batch Management     |
| Manual Expiry Tracking | Expiry Management    |
| Manual Inventory Count | ERP Inventory Count  |
| Manual Reporting       | Real-Time Dashboard  |

* Reduced stock discrepancies
* Improved inventory accuracy
